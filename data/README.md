# Experimental Data

This directory contains all experimental results and weather metadata used in the paper.

## Files

### Main Results

- **`results_model_comparison.csv`**: Performance comparison across cost models (L, E, S) for both cities and all topologies
- **`results_topology_comparison.csv`**: Detailed results for 1×1, 1×4, and 4×4 configurations
- **`monthly_trajectories.csv`**: Month-by-month cost trajectories (Figure 3 in paper)

### Ablation Studies

- **`ablation_results.csv`**: Results from ablation experiments:
  - No inter-zone temperature in state
  - No switching cost
  - No guard band
  - Train on Model L, test on Model S

### Sensitivity Analysis

- **`sensitivity_switching_cost.csv`**: Varying $c_{\text{sw}}$ from 0.05 to 0.30 SAR
- **`sensitivity_comfort_band.csv`**: Varying $[T_{\min}, T_{\max}]$ from 21-27°C to 23-25°C

### Weather Data Metadata

- **`riyadh_epw_metadata.txt`**: Summary statistics for Riyadh EPW file
- **`jeddah_epw_metadata.txt`**: Summary statistics for Jeddah EPW file

## Data Format

### results_model_comparison.csv

```csv
city,model,topology,method,cost_per_zone_day_sar,switching_freq,violations_pct,energy_kwh_per_zone_month
Riyadh,S,4x4,THERM,2.42,5.0,0.0,720
Riyadh,S,4x4,RBRL,1.98,2.5,0.0,650
...
```

### monthly_trajectories.csv

```csv
city,month,method,cost_per_zone_day_sar,cost_std_sar
Riyadh,1,THERM,1.05,0.08
Riyadh,1,RBRL,0.86,0.06
...
```

## Reproducing Results

All data files can be regenerated by running:

```bash
python experiments/train_rbrl.py --config experiments/config_riyadh.yaml --all-topologies
python experiments/evaluate_baselines.py --city riyadh --all-models
python experiments/run_ablation.py --config experiments/config_riyadh.yaml
```

## Weather Data Sources

EPW files are not included in this repository due to size constraints. Download them from:

- [EnergyPlus Weather Database](https://energyplus.net/weather)
  - Riyadh: `SAU_Riyadh.404380_IWEC.epw`
  - Jeddah: `SAU_Jeddah.410240_IWEC.epw`

Place downloaded EPW files in `data/weather/` directory.
